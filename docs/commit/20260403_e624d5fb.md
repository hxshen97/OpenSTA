# 波形传播引擎：AIG 布尔求值与事件驱动传播

- **Time**: 2026-04-03
- **Commit**: e624d5fb
- **Branch**: master

## Features

### AIG 布尔函数图
新增 AIG（And-Inverter Graph）数据结构，可将 Liberty 单元的功能表达式（FuncExpr）转换为高效的与-非图表示。支持常量节点、输入节点和与门节点，通过补码位编码实现非操作零开销。包含结构哈希去重和常量传播优化，确保图的大小最小化。

### buildAig 递归构建器
支持从 FuncExpr 递归构建 AIG，涵盖所有布尔操作：零/一常量、端口输入、非门、与门、或门（通过 De Morgan 定律）、异或门（通过 Shannon 展开）。

### 三值逻辑求值器
AIG 支持 '0'、'1'、'X' 三值逻辑求值，可对任意输入赋值计算输出值。用于在波形传播中根据输入翻转推导输出翻转。

### 事件驱动波形传播引擎
新增 WaveformPropagator，从 VCD 波形数据播种初始翻转事件，通过优先队列按时间顺序处理事件，沿组合逻辑路径传播翻转。传播过程使用 AIG 布尔求值确定输出翻转方向，通过 Liberty 弧延迟查表计算到达时间。

### 时钟沿记录与周期索引
自动识别触发器的时钟引脚翻转，记录时钟沿事件并分配周期索引，支持通过 `cycleIndexOf` 查询任意触发器在指定时间点的时钟周期。

### propagate_waveform Tcl 命令
提供 `propagate_waveform` 命令，可在 OpenSTA Tcl 环境中直接触发波形传播，支持指定 min/max 分析类型。
